一、光盘内容

    本光盘是《jQuery Mobile移动网站开发》一书的配书光盘，内容包括了书中全部实例的源程序和这些实例所使用的资源文件、程序库，以及用于演示HTML 5离线功能的配置文件等。本光盘根目录下有3个文件夹，其内容和含义说明如下。
    (1) “实例代码”文件夹：包含了《jQuery Mobile移动网站开发》中从第2到第12章的所有源代码。每章的源代码被单独安排在一个子目录中。
    (2) “简要大纲”文件夹：包含了一组PowerPoint文件。这些文件用于快速了解本书中的内容(不包括样式定义、jQuery Mobile扩展等主要通过实验方式学习的内容)。
    (3) “版权信息”文件夹：包含了本光盘所带程序库软件的版权信息。相关软件许可证信息也被包括在每一个程序库所在的子目录中。



二、运行环境

    书中出现了两种类型的网页实例：

    (1) HTML5/jQuery网页
        Windows XP/7/8，FireFox浏览器，Chrome浏览器。
    (2) jQuery Mobile网页
        Windows XP/7/8，FireFox浏览器，Chrome浏览器，Opera移动浏览器模拟器。
        Android 4.5/5，FireFox移动浏览器，Chrome移动浏览器，Opera移动浏览器。
        iOS 7/8，Safari移动浏览器。


    开发环境：

    jQuery 1.10.1，jQuery Mobile 1.4.2, Notepad++，Apache httpd服务器。
     


三、注意事项

    (1) 网页部署方法请参考本书附录A。
    (2) 有关HTML 5的离线功能测试，请参考本书第2章。
